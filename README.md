# shibe-cli-hello-world

## Description
A POC for shibe programming language.

## Tech stack
- clang
- cmake
- make
- doge

## Docker stack
- ubuntu:22.04

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
